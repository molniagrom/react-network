
### Что такое `useParams`

`useParams` — это хук в React Router, который позволяет тебе получать параметры из URL. Это удобно, когда тебе нужно извлечь данные из пути маршрута (например, ID элемента или другие параметры) и использовать их в компоненте.

### Для чего он нужен

Этот хук нужен для того, чтобы
- Получать параметры из URL.
- Использовать эти параметры в компонентах, например, чтобы отобразить информацию о пользователе, товаре или статье по ID, который указан в URL.

### Синтаксис

1. Импорт хука
   ```tsx
   import { useParams } from 'react-router-dom';
   ```

2. Использование хука
   ```tsx
   const params = useParams();
   ```
   `useParams()` возвращает объект, где каждый ключ — это имя параметра, а значение — это его значение из URL.

### Пример использования

Предположим, у нас есть URL вида `productid`, где `id` — это параметр. Мы можем получить его с помощью `useParams` и использовать для отображения данных.

```tsx
import { useParams } from 'react-router-dom';

const Product = () = {
  const { id } = useParams();   Получаем параметр id из URL

  return divProduct ID {id}div;
};
```

Если URL будет, например, `product123`, то в компоненте `Product` переменная `id` будет равна `123`.

### Когда использовать

- Когда ты хочешь, чтобы компонент был динамичным и отображал данные, связанные с параметрами URL.
- Например, когда у тебя есть список товаров и ты хочешь по ID товара показывать его подробности на другой странице.

### Примечания

- Параметры URL всегда возвращаются в виде строк, даже если это число.
- `useParams` не вызывает перерисовку компонента при изменении параметров. Если тебе нужно обновить данные на основе параметров, тебе стоит использовать эффект, например, через `useEffect`.
